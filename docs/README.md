* [ ] 과정과 미션을 출력한다.
  * [ ] 과정을 출력한다.
  * [ ] 레벨을 출력한다.
  * [ ] 미션을 출력한다.
* [ ] 과정, 레벨, 미션을 입력하라는 메시지를 출력한다.
* [ ] 과정, 레벨, 미션을 사용자로부터 입력받는다.
* [ ] 페어 매칭 작업을 한다.
* [ ] 매칭 결과를 출력한다.

* [ ] 매칭을 다시 할 지 사용하로부터 입력을 받는다.
  * [ ] 매칭하지 않는 경우 과정, 레벨, 미션 선택지로 돌아간다.
  * [ ] 매칭하는 경우 매칭 작업을 다시 한다.
